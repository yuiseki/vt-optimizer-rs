# Privacy

Purpose: Describe how vt-optimizer-rs handles data, logs, and privacy concerns.

## Data collection

- No telemetry is collected.
- The CLI does not send data to external services by default.

## Logs & diagnostics

- Output is written to stdout/stderr.
- Inputs or attributes may appear in logs when users enable detailed output.
- Users should avoid sharing logs that contain sensitive data.

## Input data considerations

- Input tiles may contain sensitive or personal information depending on the dataset.
- Users are responsible for ensuring lawful use and appropriate handling of input data.

## Data retention

- The tool does not retain data beyond the files it reads and writes locally.
